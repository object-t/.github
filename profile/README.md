# Object\<T>へようこそ🎉
## Object\<T>の概要
Object\<T>は福岡デザイン&テクノロジー専門学校の学生団体です。
学生のプログラミング力を向上させ、チーム開発経験やハッカソン経験を積むことが目的です。
[団体サイト](https://object-t.com)
